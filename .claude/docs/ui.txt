The UI is the web application that end users interact with.
Its centerpiece is a Food Comparison Table that lets anyone explore and
compare foods across nutritional and environmental dimensions side by
side.
A separate browse page lists all foods grouped by plant and animal.
Each food has its own shareable detail page.

The table is sortable by any column and columns can be toggled on and
off.

The most distinctive feature is a set of weight sliders that let users
personalize how foods are scored.
Sliders control how calorie content, protein, and serving mass factor
into scoring, alongside green water and grey water environmental impact.
A philosophical kill slider adjusts how much weight is given to animal
sentience — measured via neuron count — when scoring animal-based foods.
Scores update in real time as sliders change.

All data is fetched live from the API.
Scoring runs entirely in the browser using a WebAssembly module.
