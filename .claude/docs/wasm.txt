The wasm service provides the scoring engine that runs inside the
user's browser.
It is compiled from Rust and loaded by the web application, where it
scores and ranks foods in real time as slider values change.

Scoring covers four dimensions: greenhouse gas emissions, water
footprint (green, blue, and grey water — with green and grey being
slider-weighted), ecological impact, and a philosophical kill dimension
that weights animal sentience using neuron count as a proxy.

The module accepts the full food dataset alongside the user's current
slider settings and returns scored results ready for display.
