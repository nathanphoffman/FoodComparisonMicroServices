The data sourcing service downloads and locally caches the scientific
papers, datasets, and reference documents that the food data is derived
from.

All sources are defined in a central registry file.
The service works through each entry, downloading anything not already
saved locally.
Re-running the service is safe — it skips sources that are already
cached.

The service enforces a rate limit between requests to the same domain
and uses a full browser engine for all downloads, covering both
conventional files and JavaScript-rendered pages.
