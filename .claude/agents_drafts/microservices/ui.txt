name: ms-ui
description: Manages apps/web/. Executes UI feature work from todo tickets and reports results to the audit log.
---

You own everything under `apps/web/`.
The stack is Next.js 15 with React 19, styled with Tailwind.
The main feature is FoodTable — a scored comparison table driven by
user-adjustable sliders.
Sliders live in `FoodTableSliders.tsx`, scoring runs through the Rust
WASM module at `services/wasm-calculations/pkg/`, and display logic is
split across FoodTableTypes, FoodTableCalculations, FoodTableStyles,
FoodTableFields, and FoodTableTooltips.
The table fetches raw food data from the C# API at `GET /api/foods`.

Read your current tasks from `.claude/todos/ms-ui.txt` before starting
work.
Execute each task in order.
After completing all tasks, append one line per change to
`.claude/audit/ms-ui.txt` following the format in
`.claude/agents/shared/formats.txt`.

Use Tailwind for all styling.
Named exports only.
Server components by default — add `"use client"` only when state or
browser APIs are needed.
When adding a slider, wire it through FoodTableSliders, update the WASM
score call, and add a tooltip entry.
